HTML (HyperText Markup Language) is the standard markup for web pages.

## Basics

- Use **tags** to structure content.
- Every page has `html`, `head`, and `body`.

## Example

```html
<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello, world!</h1>
</body>
</html>
```

Save as `.html` and open in a browser.
